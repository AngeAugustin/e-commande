# 🍽️ PROJET : Manger Sain (Chez DOSSOU-YOVO) — Application Web de Gestion de Commandes

## 🧭 CONTEXTE DU PROJET

Manger Sain (Chez DOSSOU-YOVO) est un restaurant local qui souhaite digitaliser son système de commande via une application web moderne, rapide et intuitive.

L’objectif est de permettre :

* Aux clients de commander facilement sans créer de compte
* Au restaurant de gérer efficacement les commandes en temps réel

Le projet cible principalement un usage mobile (smartphones), avec une expérience fluide adaptée aux réalités locales (connexion variable, mobile money, simplicité).

---

## 🎯 OBJECTIFS PRODUIT

### 🎯 Objectifs principaux :

* Simplifier la prise de commande
* Réduire le temps d’attente
* Centraliser la gestion des commandes
* Offrir une expérience utilisateur premium

### 🎯 Contraintes :

* Pas d’authentification côté client
* Interface simple et rapide
* Compatible mobile-first
* Performances optimisées
